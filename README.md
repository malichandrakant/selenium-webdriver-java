# Selenium Login Automation

## Project Description

This project automates login functionality using Selenium WebDriver with Java.

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Chrome Browser


## Test Scenario

Test Case:
Verify user can login with valid credentials.


Steps:

1. Open login page
2. Enter username
3. Enter password
4. Click login
5. Verify successful login
